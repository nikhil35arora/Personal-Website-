# Personal Portfolio Website

A clean, responsive, multi-page personal portfolio built using **HTML5** and **CSS3**. This project serves as a digital resume and a showcase of my front-end development journey.

## 🚀 Features
* **Multi-Page Navigation:** Separate sections for Home, Education, and Personal Info.
* **Interactive UI:** Custom CSS hover effects on navigation links for better user experience.
* **Semantic HTML:** Structured for accessibility and SEO.
* **Responsive Design:** Basic layout styling designed for desktop viewing.

## 🛠️ Tech Stack
* **HTML5:** For page structure and content.
* **CSS3:** For styling, layout (Flexbox/Positioning), and animations.
* **VS Code:** Primary IDE used for development.

## 📂 Project Structure
* `index.html` - The landing page with a professional summary.
* `personal.html` - Detailed bio, skills, and hobbies.
* `education.html` - Academic background and achievements.
* `style.css` - Centralized stylesheet for all pages.
